## Context

Articulate is intended to be both working software and a public architectural journal. The project needs a durable source of truth for code, decisions, experiments, diagrams, and narrative material.

## Decision

The GitHub repository is the canonical project record. The website interprets and navigates that record, but it does not replace it.

## Consequences

Architectural material should be stored in structured, reviewable forms. The website should use content metadata and relationships to expose that material without becoming a separate content silo.
