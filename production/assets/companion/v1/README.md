# Companion v1 Runtime Asset

Expected file:

```text
production/assets/companion/v1/companion-neutral.png
```

Requirements:

- PNG with transparent background.
- One companion figure only.
- Suggested minimum dimensions: 1200 px high, with enough transparent padding for contain-style placement.
- Half-body or bust framing, neutral opening pose.
- No labels, logo, embedded text, design-board marks, neighbouring poses or reference annotations.
- Natural standing or presenter posture; head and hands must not be cropped.
- Consistent lighting and colour with the Companion PoC palette.
- Standard RGB colour space is preferred.

Validation checklist:

- The file opens as a PNG.
- The background is transparent.
- The image contains exactly one figure.
- The figure remains recognisable when contained inside a 650 x 850 px region.
- The asset is not the full character reference sheet.
