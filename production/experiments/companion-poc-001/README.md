# Companion POC 001

Experiment status: draft vertical slice

Canonical source: [`docs/episodes/0001-why-articulate-exists.md`](../../../docs/episodes/0001-why-articulate-exists.md)

Related production workspace: [`production/episodes/0001/`](../../episodes/0001/)

## Purpose

This experiment tests whether a short section of Episode 0001 can become a coherent narrated visual experience without requiring Russell to appear on camera.

It is not the full Episode 0001 video. It does not select a permanent avatar vendor, lip-sync provider, asset versioning standard or video assembly framework.

## Prototype Shape

Target duration: 60-90 seconds

| Time | Purpose | Primary mode |
| --- | --- | --- |
| 00:00-00:14 | Companion-led opening and disclosure | Companion presenter |
| 00:14-00:38 | Why understanding architecture remains hard | Visual narrative |
| 00:38-01:05 | Fragmented knowledge and evolving body of knowledge | Concept diagram |
| 01:05-01:20 | Return to companion and close | Companion presenter |

## Experiment Questions

- Can the written architectural argument survive adaptation into a short spoken script?
- Does the companion clarify pacing without appearing to be Russell or the author?
- Can a non-slide visual sequence make the architectural problem visible?
- Which rendering, lip-sync, asset and assembly constraints appear before vendor selection?
- What must human review check before a rough cut is trusted?

## Boundaries

- Russell remains the author, researcher, narrator and accountable reviewer.
- The companion is an AI-created visual presenter, not a digital clone or autonomous author.
- All material claims must trace to the canonical episode or be labelled as editorial additions.
- Generated visuals are review artefacts until approved.
- This workspace records intent, prompts and evaluation criteria; it does not contain final media.
