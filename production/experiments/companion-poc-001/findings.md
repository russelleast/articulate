# Findings

Experiment status: planned

## Initial Findings

- Episode 0001 already contains a complete long-form production plan. This experiment should remain a narrow slice, not a replacement production plan.
- The strongest short-form source arc is Introduction -> The Reality of Modern Architecture -> The Vision.
- Direction B, Architectural Companion, is the best first candidate because it supports guide-like continuity without making the companion feel like the author.
- Lip-sync is a constraint to test, not a prerequisite. A still or lightly animated companion may be enough for the first evidence-producing rough cut.

## Constraints To Validate

- Companion consistency across opening and closing scenes.
- Whether the disclosure is sufficient and unobtrusive.
- Whether generated fragmented-knowledge visuals introduce unsupported claims.
- Whether the concept diagram can stay exploratory rather than implying Articulate already solves the problem.
- Whether the asset register is detailed enough before binary assets exist.

## Candidate ADR Topics

- Companion rendering boundary and anti-impersonation requirements.
- Generated asset provenance and versioning.
- Minimum review package for AI-assisted media experiments.
- Lip-sync policy for AI-created presenters.

## Next Review Step

Run the evaluation test plan against these draft artefacts before generating any visual assets or assembling a rough cut.
