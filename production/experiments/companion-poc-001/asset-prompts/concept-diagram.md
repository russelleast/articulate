# Concept Diagram Prompt

Purpose: produce the central visual transformation for S004.

## Prompt

Design a simple animated concept diagram for an architectural essay. Start with disconnected document fragments labelled source code, IaC, ADRs, diagrams, slides, monitoring and memory. Transform them into a connected knowledge graph around the question: "A continuously evolving body of knowledge?" The diagram should remain unresolved and exploratory, not claim the system already exists. Use restrained architectural notation, clear spacing, readable labels and subtle motion.

## Diagram Rules

- Use only concepts grounded in Episode 0001.
- Do not add vendor logos, model names, timelines or product claims.
- Keep the central question visually dominant.
- Show movement from fragments towards connection, but stop short of a solved state.

## Accessibility Notes

- Labels must remain readable at 1080p.
- The narration must explain the visual transformation.
- Avoid colour-only meaning.
