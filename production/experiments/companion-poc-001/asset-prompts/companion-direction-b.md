# Companion Prompt - Direction B

Purpose: generate still-frame candidates for Direction B, Architectural Companion.

## Prompt

Create a calm AI-generated visual companion for an architectural essay video. The companion is a semi-human designed presenter, clearly artificial but warm and credible, with restrained gestures, neutral contemporary clothing, and a quiet research-studio setting. The image should suggest architectural thinking, source traceability and diagrams without looking like a corporate training video. The companion should guide the viewer beside journal pages and concept diagrams, not appear to be the author. Medium shot, eye line to camera, simple background with subtle document and diagram elements, balanced contrast, no dramatic lighting.

## Negative Guidance

Avoid photorealistic human impersonation, Russell likeness, mascot styling, exaggerated smile, cyberpunk, glossy sales-video finish, dense UI panels, floating holograms, purple-blue gradient background, cluttered desk.

## Review Checks

- Feels like a guide, not an autonomous expert.
- Has enough visual identity to recur.
- Leaves visual attention available for source material.
- Supports disclosure that it is AI-created.
