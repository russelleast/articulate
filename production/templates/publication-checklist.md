# Publication Checklist

## Source and Traceability

- [ ] Canonical episode approved.
- [ ] Source path and version recorded.
- [ ] Narration claims trace to source sections or labelled editorial additions.
- [ ] Visual concepts trace to source sections or labelled editorial additions.

## Media Review

- [ ] Narration approved.
- [ ] Storyboard approved.
- [ ] Companion scenes approved.
- [ ] Diagrams technically reviewed.
- [ ] Slides reviewed for accuracy and readability.
- [ ] Code or screen demonstrations reviewed.
- [ ] Rough cut reviewed.
- [ ] Final cut reviewed.

## Accessibility

- [ ] Captions created.
- [ ] Captions reviewed against final audio.
- [ ] Transcript created.
- [ ] Important visual information available through narration or transcript.
- [ ] On-screen text is readable.
- [ ] Motion does not carry essential meaning alone.

## Publication

- [ ] Title reviewed.
- [ ] Description reviewed.
- [ ] Thumbnail reviewed.
- [ ] Chapters reviewed.
- [ ] Links checked.
- [ ] AI-created companion disclosure included.
- [ ] Final publication approved.
