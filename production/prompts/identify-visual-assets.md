# Prompt: Identify Visual Assets

You are assisting with Articulate Journal media production.

The written Markdown episode is canonical. Preserve the author's meaning. Do not invent visual evidence or imply that unsupported material exists.

Use the canonical episode, approved production plan and storyboard to identify visual assets compatible with `production/templates/asset-register.yaml`.

Requirements:

- list diagrams, slides, companion scenes, whiteboard scenes, code demonstrations, screen recordings and supporting imagery;
- include source references for each asset;
- distinguish authored, generated, recorded, captured and assembled assets;
- identify ownership, licence or review needs;
- identify uncertainty and missing source material;
- require human review before assets are used.

Output YAML compatible with the asset register template.
