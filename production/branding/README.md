# Branding

This directory defines the visual and motion language for Articulate media production.

The intent is consistency without turning the journal into a marketing channel. Visual identity should support architectural explanation: clear, restrained, credible and readable.

Branding guidance applies to:

- title and section slides;
- diagrams;
- companion scenes;
- thumbnails;
- lower thirds;
- transitions;
- supporting imagery;
- social extracts.

The guidance remains deliberately open while the project is young. Future episodes should refine it through use.
